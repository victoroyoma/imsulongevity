# IMSU International Conference on Longevity 2026

Official website for the IMSU International Conference on Longevity 2026, themed **"Biophilosophy, Afrolongevity & Human Enhancement"** - A pioneering conference exploring aging, longevity, and human enhancement from an African perspective.

## 🎯 About the Conference

The conference brings together global thought leaders, researchers, policymakers, and practitioners to address longevity challenges and opportunities in Africa. It focuses on:

- Biological and clinical sciences of aging
- Technology and innovation in longevity
- Pharmaceutical and drug safety
- Nutrition and food systems
- Ethics and philosophy of human enhancement
- Public health and healthcare systems

**Date:** June 24-25, 2026  
**Venue:** University Auditorium, Imo State University, Owerri, Nigeria

## ✨ Features

- **Responsive Design**: Fully optimized for mobile, tablet, and desktop devices
- **Interactive UI**: Smooth animations and transitions using Framer Motion
- **Registration System**: Modal-based registration forms for conference attendance, paper submissions, and abstract submissions
- **Dynamic Navigation**: Active section tracking with smooth scroll and mobile menu
- **Google Maps Integration**: Interactive venue location map
- **Accessibility**: WCAG compliant with proper ARIA labels, focus states, and keyboard navigation
- **Modern Tech Stack**: Built with React, TypeScript, and Tailwind CSS

## 🛠️ Tech Stack

- **Framework**: React 18 with TypeScript
- **Build Tool**: Vite
- **Styling**: Tailwind CSS with custom design tokens
- **Animations**: Framer Motion
- **Icons**: Lucide React
- **Fonts**: Inter (body), Playfair Display (headings)

## 📦 Installation

1. **Clone the repository**
   ```bash
   git clone <repository-url>
   cd imsulongevity
   ```

2. **Install dependencies**
   ```bash
   npm install
   ```

3. **Start development server**
   ```bash
   npm run dev
   ```

4. **Open in browser**
   ```
   http://localhost:5173
   ```

## 🎨 Components Overview

### Navigation
- Fixed navigation with scroll-based styling changes
- Active section tracking with visual indicators
- Mobile-responsive hamburger menu with backdrop overlay
- Integrated registration modal trigger

### Hero
- Eye-catching landing section with conference branding
- Multiple CTAs (Register, Call for Papers, Submit Abstract)
- Responsive typography and spacing
- Animated elements with Framer Motion

### Registration Modal
- Three types: Conference registration, paper submission, abstract submission
- Form validation and success states
- Scrollable content for long forms
- Mobile-optimized input fields

### Call for Papers
- Six main categories with 24+ sub-themes
- Important dates timeline
- Submission guidelines
- Comprehensive concept note section

### Speakers
- Categorized speaker display (Keynote, Lead Presenter, Special Guest)
- Interactive speaker cards with hover effects
- Responsive grid layout
